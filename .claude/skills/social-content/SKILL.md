# Social Content

Write content that is native to each platform. The same idea is packaged
differently depending on where it lives.

## The hook is everything

The first line decides whether anything else gets read.

- Lead with the payoff, tension, or a concrete number — not a warm-up.
- Make a specific promise or provoke curiosity; avoid vague "thread 🧵" teases.
- Test 3–5 hook variants for any important post.

Hook patterns: contrarian take · specific result ("$0 → $10k in 60 days") ·
list promise ("7 things I wish I knew") · question that names the reader's pain ·
"Most people get X wrong. Here's why."

## Platform playbook

**X / Twitter**
- Short lines, lots of white space. One idea per tweet.
- Threads: hook tweet must stand alone; each tweet earns the next.
- End with a soft CTA (follow, reply, bookmark) only when value is delivered.

**LinkedIn**
- First 1–2 lines visible before "see more" — front-load the hook.
- Personal story → lesson → takeaway. Short paragraphs, generous line breaks.
- Professional but human; avoid corporate jargon. End with a question.

**Instagram**
- Visual-first; caption supports the image/carousel.
- Carousels: cover slide is the hook, each slide one point, last slide CTA.
- Caption opens with a hook line; keep it scannable.

**TikTok / Reels / Shorts (scripts)**
- Hook in the first 1–2 seconds (verbal + on-screen text).
- Structure: hook → deliver value fast → quick payoff → loop or CTA.
- Write for the ear: short, spoken sentences. Pattern interrupts every few sec.

**YouTube**
- Title + thumbnail do 80% of the work; promise a clear outcome.
- Open by restating the promise and previewing the payoff to retain viewers.

**Newsletter**
- Subject line = the hook; preview text reinforces it.
- One core idea per send; scannable with subheads; one clear CTA.

## Voice & quality

- Write like a person talks. Cut filler, hedging, and throat-clearing.
- Be specific: numbers, names, concrete examples beat abstractions.
- One idea per post. If it needs two, make two posts.
- Show, don't claim — examples and stories over adjectives.

## CTA principles

- One CTA per piece. Make the next action obvious and low-friction.
- Earn the ask: deliver value first, then invite the next step.

## Repurposing flow

Long-form → extract 3–5 atomic ideas → write each as a platform-native post →
adapt format per platform → vary hooks. One asset becomes a week of content.
